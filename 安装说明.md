## 🚀 快速安装（Windows）

### 方法一：使用安装脚本
```cmd
# 进入安装包目录，双击运行 install.bat
# 或者在命令行执行：
install.bat
```

### 方法二：手动安装
```cmd
# 1. 创建全局技能目录
mkdir %USERPROFILE%\.agents\skills\技能管理大师

# 2. 复制所有文件到该目录
copy SKILL.md %USERPROFILE%\.agents\skills\技能管理大师\
copy scripts\* %USERPROFILE%\.agents\skills\技能管理大师\

# 3. 运行更新脚本初始化技能表
python %USERPROFILE%\.agents\skills\技能管理大师\update_skill_table.py
```

## 📖 功能说明

技能管理大师提供以下核心功能：

### 1. 技能自动扫描
- 扫描全局技能目录：`~/.agents/skills/`
- 扫描项目本地技能目录：`./.trae/skills/`
- 解析 SKILL.md 的 frontmatter 获取技能信息

### 2. 技能表管理
- JSON 格式：`skill_table.json`（便于程序读取）
- Markdown 格式：`技能管理表.md`（便于人工阅读）
- 自动分类：AI功能类、开发工具类、测试类、云服务类等

### 3. 同步检查
- 检测新增技能
- 检测删除技能
- 检测技能描述变更
- needs_sync 标志提醒更新

### 4. 需求优先分析
- 每次调用时先理解用户需求
- 分析任务复杂度和所需技能类型
- 评估是否真的需要调用技能（避免过度工程化）

## 🛠️ 自定义配置

### 修改分类关键词
编辑 `scripts/update_skill_table.py` 中的关键词列表：

```python
ai_keywords = ['ai', 'image', 'face', 'generate', ...]
dev_keywords = ['code', 'develop', 'design', ...]
# 添加或修改关键词以适应您的需求
```

### 修改扫描目录
编辑 `scripts/scan_skills.py` 添加更多扫描路径：

```python
# 在 scan_skills() 函数中添加
custom_skills_dir = Path('your/custom/path')
installed_skills.extend(scan_directory(custom_skills_dir))
```

## ❓ 常见问题

### Q: 安装后找不到技能？
A: 确保文件复制到了正确的位置：`%USERPROFILE%\.agents\skills\技能管理大师\`

### Q: Python 脚本无法运行？
A: 确保已安装 Python 和 PyYAML，可运行 `pip install pyyaml` 重新安装依赖

### Q: 如何卸载？
A: 直接删除 `%USERPROFILE%\.agents\skills\技能管理大师\` 目录即可

2. PyYAML 是否正确安装
3. 文件路径是否包含中文字符（确保编码为 UTF-8）
